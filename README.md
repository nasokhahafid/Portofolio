# Portfolio Project

This is a personal portfolio website built with Laravel 9, Tailwind CSS, and Vite. It features a modern, responsive design with smooth animations.

## Features

-   **Hero Section**: Introduction and welcome message.
-   **About Section**: Information about the developer.
-   **Experience Section**: Timeline of professional experience.
-   **Skills Section**: Technical skills showcase.
-   **Certificates Section**: Display of certifications.
-   **Contact Form**: Functional contact form (backend implementation ready).
-   **Responsive Design**: Optimized for mobile and desktop.
-   **Dark Mode**: Stylish dark theme aesthetics.
-   **Animations**: AOS (Animate On Scroll) integration.

## Tech Stack

-   **Backend**: Laravel 9 (PHP 8.2+)
-   **Frontend**: Blade Templates, Tailwind CSS 3
-   **Database**: MySQL
-   **Build Tool**: Vite

## Setup

1.  Clone the repository.
2.  Run `composer install`.
3.  Run `npm install`.
4.  Copy `.env.example` to `.env` and configure your database.
5.  Run `php artisan migrate --force`.
6.  Run `npm run build` or `npm run dev`.
7.  Run `php artisan serve`.
